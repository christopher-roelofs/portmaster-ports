
**Controls**

Start/Select = Pause

Start+Select = Exit

Dpad  and Left stick = left/right and look up/down
Right Stick = left/right

A = Jump
B = Dash
Y = Hold to reset
X = Toggle screenshake

l1/l2 = Load state
r1/r2 - Save state

Start + Select = Exit
